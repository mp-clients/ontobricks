run_inference(data_turtle, rule_shapes, base_uri=None)[source]¶

Run SHACL-AF inference rules and return newly inferred triples.

Uses pyshacl.validate(advanced=True, inplace=True) to execute sh:TripleRule and sh:SPARQLRule shapes. The data graph is expanded in-place and then diffed against the original to extract the inferred triples.
